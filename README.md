
## Запуск

```bash
Создай .env файл и добавь DATABASE_URL
$ npm install

стоит postinstall хук на prisma generate чтобы сгенерировать клиент призмы
```

## Запуск

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Тесты

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
